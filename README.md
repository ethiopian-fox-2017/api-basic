# api-basic
## REST API

list of user routes

|route         | http | description                |
|:-------------|:-----|:-----------                |
|api/users     | GET  | get data all users         |
|api/user/:id  | GET  | get single data            |
|api/user      | POST | insert new data            |
|api/user/:id  | DELETE| DELETE DATA BASE ON ID    |
|api/user/:id  | PUT  | update data                |

# Usage
npm install
npm start
