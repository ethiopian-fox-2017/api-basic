# api-basic

Demo app with basic REST API.

## How to use

```
> npm install
> node app.js
```


## List of Route

#### Api Store Route

| Route                               | HTTP   | Description                             |
| ----------------------------------- | ------ | --------------------------------------- |
| api/users                          | GET    | Show all users                               |
| api/users/:id                       | GET    | Show user by id                        |
| api/users                           | POST   | Create user                            |
| api/users/:id                           | PUT    | Update user                            |
| api/users/:id                           | DELETE | Delete User                            |
